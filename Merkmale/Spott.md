# `=this.file.name`
Jede feindliche Kreatur innerhalb von 18 Metern (12 Kästchen) in Hör- oder Sichtweite muss einen [[Rettungswurf]] auf [[Charisma]] bestehen.
Bei Misserfolg erleidet sie [[Vorteil und Nachteil|Nachteil]] auf alle [[Angriff|Angriffe]] die eine andere als die auslösende Kreatur als Ziel haben.
Danach kann sie einen [[Angriff]] als [[Bonusaktion]] ausführen.
Dieser Effekt endet zu Beginn des nächsten [[Zug|Zuges]] der auslösenden Kreatur.
Dieses Merkmal kann nur einmal pro Tag eingesetzt werden.
