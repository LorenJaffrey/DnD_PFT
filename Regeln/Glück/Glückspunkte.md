# `=this.file.name`

[[Glückspunkte]] können vom DM verliehen werden wenn ein Spieler besonders im Einklang mit seinem Charakter handelt. 
Das gilt vor allem wenn der Spieler die [[Persönliche Charakteristika]] ausspielt.
Spieler können beliebig viele [[Glückspunkte]] sammeln.
