# `=this.file.name`

`=this.file.name` wird in der Mengeneinheit [[Ration]] verwaltet. Eine Ration entspricht dabei 1 Pfund `=this.file.name`.
Ein Charakter benötigt 1 [[Ration|Ration]] pro Tag und kann länger damit auskommen, indem er nur halbe [[Ration]] isst.

## Bedarf an `=this.file.name`
| Kreaturengröße |      Bedarf pro Tag      |
| -------------- |:------------------------:|
| Winzig         | 1/4 [[Ration|Ration]] |
| Klein          |  1 [[Ration|Ration]]  |
| Mittel         |  1 [[Ration|Ration]]  |
| Groß           |      4 [[Ration]]      |
| Sehr groß      |     16 [[Ration]]      |
| Gigantisch     |     64 [[Ration]]      |

## Aufnahme von `=this.file.name`  reduzieren
Reduziert ein Charakter die zu sich genommene `=this.file.name` auf die Hälfte, zählt jeder Tag als halber Tag ohne Essen. 

Steht dem Spieler an einem Tag keine `=this.file.name` zur Aufnahme zur Verfügung, so muss er einen [[Rettungswurf]] auf [[Konstitution]] gegen [[Schwierigkeitsgrad|SG]] 10 ablegen.
Schlägt die Probe fehl, erhält der Charakter eine Stufe [[Erschöpft|Erschöpfung]].
