# `=this.file.name`

Die Bewegungsgrate wird meist durch die [[Spezies]] festgelegt und beträgt in den meisten Fällen 9m pro [[Zug]].
Eine Kreatur kann auch eine eigene Bewegungsrate für  bestimmte Fortbewegungsarten wie [[Schwimmen]], [[Fliegen]], [[Graben]], etc. besitzen.

## Bewegungsarten
Bestimmte Bewegungsarten haben eine Auswirkung auf die [[Bewegungsrate]].
