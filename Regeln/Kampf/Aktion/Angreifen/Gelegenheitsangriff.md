# `=this.file.name`

Wenn eine Kreatur die Nahkampfreichweite  einer anderen Kreatur verlässt, kann diese als [[Reaktion]] einen `=this.file.name` ausführen.
`=this.aliases[0]` erhalten keinen positiven [[Attribute#Attributsmodifikator]].
`=this.aliases[0]` können durch [[Rückzug]] verhindert werden.
