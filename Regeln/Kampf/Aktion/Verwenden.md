# `=this.file.name`
Normalerweise interagierst du im Rahmen einer anderen [[Aktion]] mit Gegenständen - wenn du beispielsweise im Rahmen der [[Angreifen|Angriffsaktion]] das Schwert ziehst. 
Wenn eine [[Aktion]] erforderlich ist, um einen Gegenstand zu verwenden, führst du die [[Verwenden]]-Aktion aus.
Bei Benutzung von magischen Gegenständen ist immer eine [[Verwenden]]-Aktion notwendig.
