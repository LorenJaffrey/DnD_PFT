# `=this.file.name`

Eine Kreatur kann normalerweise die nachfolgenden [[Aktion|Aktionen]] ausführen.
Sie kann jedoch gegebenenfalls weitere Aktionen anhand ihrer [[Klassen|Klasse]] oder ihres Statblock zur Verfügung haben.
