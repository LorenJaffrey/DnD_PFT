# `=this.file.name`
Wenn du diese [[Aktion]] durchführst, machst du einen Wurf auf [[Weisheit]], um etwas zu erkennen, was nicht offensichtlich ist. 
Die folgende Tabelle zeigt an, welche [[Fertigkeiten]] anwendbar sind, wenn du diese [[Aktion]] durchführst, abhängig davon, was du zu entdecken versuchst. 

| Fähigkeit           | Zu erkennendes Objekt         |
| ------------------- | ----------------------------- |
| [[Motiv erkennen]]  | Gemütszustand einer Kreatur   |
| [[Heilkunde]]       | Leiden einer Kreatur          |
| [[Wahrnehmung]]     | Verdeckte Kreatur oder Objekt |
| [[Überlebenskunst]] | Spuren oder Nahrung           |
