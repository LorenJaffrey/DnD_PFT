# `=this.file.name`

Eine Kreatur kann nur [[Fliegen]] wenn sie eine extra [[Bewegungsrate]] dafür hat.

Wenn fliegende Kreatur zu Boden gestoßen, [[Bewegungsrate]] auf 0 reduziert, oder auf andere Weise bewegungsunfähig gemacht wird, stürzt sie ab (siehe [[Stürzen]]).
Dies wird nur verhindert wenn Kreatur [[Schweben]] kann oder durch Magie in der Luft gehalten wird.
