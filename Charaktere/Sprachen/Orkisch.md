# `=this.file.name`

[[Orkisch]] ist eine harsche, gutturale Sprache, die kein geschriebenes Skript hat. 
Wenn sie zum Schreiben aufgerufen werden, verwenden sie eine phonetische Version ihrer Sprache, kombiniert mit der [[Zwergisch|zwergischen]] Schrift anstelle eines tatsächlichen Alphabets, das speziell für die Sprache der Orks verwendet wird.
